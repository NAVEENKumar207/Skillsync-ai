### 🚀 SYSTEM STATUS: OPERATIONAL
**SkillSync AI** is a high-performance career optimization platform designed to bridge the gap between your current data points and your dream terminal role. Powered by the **Groq Llama-3.3-70B** core, it executes deep analysis of your resume against the mainframe standards of Tier-1 tech entities (Google, Meta, Amazon, etc.).

---

### ✨ HARDWARE FEATURES

- **[P-01] DEEP PARSING ENGINE** 📄  
  Accepts .PDF, .DOCX, and .TXT inputs. Automatically redacts sensitive metadata (PII) before analysis.
  
- **[P-02] TARGET BENCHMARKING** 🏢  
  Cross-references your profile against specific corporate logic gates for elite engineering roles.

- **[P-03] 3-MONTH UPGRADE ROADMAP** 🗺️  
  Generates a week-by-week, day-by-day protocol for:
  - **M1:** Core Foundation Rewriting
  - **M2:** Production-Grade Implementation
  - **M3:** System Design & Behavioral Mastery

- **[P-04] INTEGRATED AI ASSISTANT** 🤖  
  A persistent, Groq-powered chat module for on-the-fly career debugging.

### 🛠️ CORE ARCHITECTURE (STACK)

| MODULE | COMPONENT |
| :--- | :--- |
| **USER INTERFACE** | React.js, Tailwind CSS (Retro-Bold), Framer Motion |

| **DATA STORAGE** | MongoDB Atlas (NoSQL) |

| **SERVICES** | Nodemailer (Recovery), PDF-Parse (Extraction) |

### 🕹️ INITIALIZATION SEQUENCE

1. **CLONE SOURCE**
   ```bash
   git clone https://github.com/NAVEENKumar207/Skillsync-ai.git
   ```

2. **BOOT SERVER**
   ```bash
   cd server && npm install && npm start
   ```

3. **BOOT CLIENT**
   ```bash
   cd client && npm install && npm start
   ```

---

### 🔐 ENVIRONMENT OVERRIDE (.env)

Configure your `server/.env` with these critical constants:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
GROQ_API_KEY=your_groq_api_key
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
```
